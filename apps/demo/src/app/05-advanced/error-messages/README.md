# Error Messages & Global Configuration (archived reference)

This standalone error-message demo is no longer routed.

Its core teaching goal — the toolkit's message-resolution priority and local/global configuration interplay — is now represented in `advanced-scenarios/global-configuration` and in the package-level documentation.

## Prefer these current references instead

- `advanced-scenarios/global-configuration`
- `docs/PARSE_ERRORS_AND_WARNINGS.md`

## Why this page was merged

Keeping message priority and global defaults on separate routes created unnecessary overlap. The consolidated `global-configuration` example is now the primary place to study:

- global defaults
- per-form overrides
- deterministic control identity
- message resolution and wrapper/debugger behavior
